# EMG肌电信号采集设备与信号基础

## 第一部分：EMG信号的生理学基础

### 1.1 肌肉收缩的生理机制

#### 1.1.1 神经肌肉系统
人体运动是通过中枢神经系统控制骨骼肌收缩实现的。完整的神经-肌肉控制链包括：

1. **大脑运动皮层**：产生运动意图，发出运动指令
2. **脊髓传导**：运动神经元将指令传递到外周
3. **神经肌肉接头**：运动神经元与肌纤维的连接点
4. **肌肉收缩**：肌纤维接收信号后产生收缩

#### 1.1.2 运动单位（Motor Unit, MU）
运动单位是神经肌肉系统的基本功能单元，包括：
- 一个α运动神经元
- 该神经元支配的所有肌纤维（几个到数千个）

关键特性：
- 同一运动单位的所有肌纤维同时激活
- 不同运动单位的募集遵循"大小原理"（Size Principle）
- 小运动单位先募集（产生精细动作）
- 大运动单位后募集（产生强力收缩）

#### 1.1.3 动作电位的产生
肌肉收缩时产生电信号的过程：

1. **静息状态**
   - 细胞膜内外存在电位差（约-70mV）
   - 膜内负电荷，膜外正电荷

2. **去极化过程**
   - 神经递质（乙酰胆碱）作用于肌纤维膜
   - Na+通道开放，Na+内流
   - 膜电位从-70mV升至+30mV

3. **复极化过程**
   - K+通道开放，K+外流
   - 膜电位恢复至静息水平

4. **动作电位传播**
   - 沿肌纤维膜双向传播
   - 速度约3-5 m/s

### 1.2 EMG信号的定义与分类

#### 1.2.1 EMG信号定义
肌电图（Electromyography, EMG）是记录肌肉激活时产生的生物电信号的技术。这些电信号是多个运动单位动作电位（Motor Unit Action Potential, MUAP）在时间和空间上叠加的结果。

#### 1.2.2 EMG信号分类

**按采集方式分类：**

1. **表面肌电（Surface EMG, sEMG）**
   - 采集方式：电极贴在皮肤表面
   - 优点：无创、操作简单、适合动态监测
   - 缺点：空间分辨率低、容易受干扰
   - 应用：运动分析、康复评估、人机交互

2. **针电肌电（Needle EMG）**
   - 采集方式：针状电极插入肌肉内部
   - 优点：高空间分辨率、信号质量高
   - 缺点：侵入性、不适合长时间监测
   - 应用：临床诊断、神经肌肉疾病评估

**按信号性质分类：**

1. **原始EMG（Raw EMG）**
   - 未经处理的原始电信号
   - 幅度：50μV - 5mV
   - 频率范围：20-500 Hz

2. **处理后EMG**
   - 滤波EMG：去除噪声和干扰
   - 整流EMG：取绝对值
   - 包络EMG：平滑处理
   - RMS EMG：均方根值

### 1.3 EMG信号的基本特征

#### 1.3.1 时域特征

**幅度特性：**
- 静息状态：5-20 μV（基线噪声）
- 轻度收缩：50-200 μV
- 中度收缩：200-500 μV
- 强力收缩：500 μV - 5 mV
- 最大自主收缩（MVC）：可达5-10 mV

**时间特性：**
- 随机性：类似高斯白噪声
- 非平稳性：统计特性随时间变化
- 突发性：肌肉收缩时信号突然增大
- 零均值：长时间平均接近零

**统计特性：**
- 概率分布：近似高斯分布
- 自相关：快速衰减，表明信号的随机性
- 互相关：不同通道间存在一定相关性

#### 1.3.2 频域特征

**频谱分布：**
- 主要能量范围：20-500 Hz
- 能量集中区域：50-150 Hz
- 峰值频率：通常在80-120 Hz
- 频谱形状：随肌肉类型和收缩强度变化

**频率成分来源：**
- 低频成分（<20 Hz）：运动伪影、基线漂移
- 主频段（20-150 Hz）：肌肉激活信号
- 高频成分（150-500 Hz）：快速运动单位、表层肌肉
- 超高频（>500 Hz）：噪声、电极接触问题

**影响因素：**
1. **肌肉类型**
   - 快肌纤维：频谱偏高频
   - 慢肌纤维：频谱偏低频

2. **收缩强度**
   - 强度增加 → 更多运动单位募集
   - 高频成分增加

3. **肌肉疲劳**
   - 疲劳时频谱向低频移动
   - 平均频率（MNF）和中值频率（MDF）下降
   - 这是疲劳检测的重要指标

#### 1.3.3 EMG信号与力量的关系

**EMG-力量关系：**
- 低强度收缩（<30% MVC）：近似线性关系
- 中高强度收缩：非线性关系
- 数学模型：
  ```
  RMS(EMG) = a × Force^b
  其中 a, b 是经验参数
  ```

**影响因素：**
- 肌肉长度：肌肉在最优长度时EMG最大
- 收缩类型：等长、向心、离心收缩特性不同
- 疲劳状态：疲劳时需要更高EMG产生相同力量

## 第二部分：EMG信号采集系统

### 2.1 采集系统总体架构

完整的EMG采集系统包括以下环节：

```
肌肉 → 电极 → 前置放大器 → 滤波器 → ADC → 数据处理 → 存储/显示
```

每个环节的作用：
1. **电极**：将离子电流转换为电子电流
2. **前置放大器**：放大微弱信号，提高信噪比
3. **滤波器**：去除不需要的频率成分
4. **ADC**：模拟信号数字化
5. **数据处理**：特征提取、模式识别
6. **存储/显示**：数据管理和可视化

### 2.2 电极系统

#### 2.2.1 电极类型与特性

**1. 表面电极（Surface Electrodes）**

**湿电极（Wet/Gel Electrodes）：**
- 材料：Ag/AgCl（银/氯化银）
- 优点：
  - 阻抗低（1-10 kΩ）
  - 信号质量好
  - 价格便宜
- 缺点：
  - 需要皮肤预处理
  - 导电膏容易干燥
  - 长时间使用不舒适
- 应用：实验室研究、临床评估

**干电极（Dry Electrodes）：**
- 材料：不锈钢、金属合金
- 优点：
  - 无需导电膏
  - 使用方便
  - 可重复使用
- 缺点：
  - 阻抗较高（10-100 kΩ）
  - 易受运动伪影影响
- 应用：可穿戴设备、长期监测

**织物电极（Textile Electrodes）：**
- 材料：导电纤维、碳纳米管
- 优点：
  - 可集成到衣物
  - 舒适性好
  - 适合日常穿戴
- 缺点：
  - 信号质量不稳定
  - 耐用性问题
- 应用：智能服装、运动监测

**2. 针电极（Needle Electrodes）**
- 类型：单针、同心针、双极针
- 应用：临床神经肌肉诊断
- 本课程不涉及

#### 2.2.2 电极配置

**单极配置（Monopolar）：**
```
记录电极（肌肉上方） → 放大器 ← 参考电极（远端）
```
- 优点：简单
- 缺点：容易受共模干扰

**双极配置（Bipolar）：**
```
电极1 ─┐
       ├→ 差分放大器 → 输出
电极2 ─┘
```
- 优点：共模抑制比高，抗干扰能力强
- 缺点：只能检测电极间的电位差
- 应用：最常用配置

**多通道阵列（Multi-channel Array）：**
```
电极1  电极2  电极3  电极4
  │      │      │      │
  └──────┴──────┴──────┘
         放大器阵列
```
- 优点：空间分辨率高，可分析肌肉激活模式
- 缺点：系统复杂，成本高
- 应用：高精度研究、假肢控制

#### 2.2.3 电极放置原则

**SENIAM标准（Surface ElectroMyoGraphy for the Non-Invasive Assessment of Muscles）：**

1. **位置选择**
   - 肌腹中央：避开肌腱和神经肌肉接头
   - 肌纤维方向：电极连线平行于肌纤维
   - 避开脂肪层厚的区域

2. **电极间距**
   - 标准间距：20mm（中心距）
   - 小肌肉：10-15mm
   - 大肌肉：20-30mm
   - 间距过小：空间分辨率低
   - 间距过大：信号幅度降低

3. **皮肤预处理**
   - 清洁：酒精擦拭
   - 去角质：轻微打磨（降低阻抗）
   - 剃毛：必要时去除毛发
   - 目标阻抗：<10 kΩ

4. **常见肌肉电极位置（手部）**
   - 指浅屈肌（FDS）：前臂掌侧近端1/3
   - 指深屈肌（FDP）：前臂掌侧中段
   - 桡侧腕屈肌（FCR）：前臂掌侧近端
   - 尺侧腕屈肌（FCU）：前臂掌侧内侧
   - 拇长展肌（APL）：前臂背侧远端

### 2.3 信号调理电路

#### 2.3.1 前置放大器

**差分放大器原理：**
```
V_out = G × (V+ - V-)
```
其中：
- V+：正输入（电极1）
- V-：负输入（电极2）
- G：增益（1000-10000）

**关键指标：**

1. **共模抑制比（CMRR）**
   - 定义：差模增益与共模增益之比
   - 公式：CMRR = 20 × log10(Ad / Ac) dB
   - 要求：CMRR > 80 dB
   - 意义：抑制共模干扰（如工频干扰）

2. **输入阻抗（Input Impedance）**
   - 要求：>10 MΩ
   - 原因：
     - 电极-皮肤阻抗高（1-100 kΩ）
     - 高输入阻抗避免信号衰减
     - 减少电极阻抗不匹配的影响

3. **噪声水平（Input Noise）**
   - 要求：<5 μV（rms）
   - EMG信号最小幅度约50 μV
   - 信噪比要求：SNR > 20 dB

4. **增益（Gain）**
   - 典型值：1000-10000
   - 选择依据：
     - EMG幅度：50 μV - 5 mV
     - ADC输入范围：±5V
     - 增益 = ADC范围 / EMG最大幅度

#### 2.3.2 滤波器

**硬件滤波器（Analog Filters）：**

1. **高通滤波器（HPF）**
   - 截止频率：10-20 Hz
   - 作用：
     - 去除运动伪影（<10 Hz）
     - 去除基线漂移
     - 去除ECG干扰（心电）
   - 类型：巴特沃斯（Butterworth）

2. **低通滤波器（LPF）**
   - 截止频率：400-500 Hz
   - 作用：
     - 防止混叠（Anti-aliasing）
     - 去除高频噪声
     - 满足奈奎斯特采样定理
   - 类型：贝塞尔（Bessel）或巴特沃斯

3. **陷波滤波器（Notch Filter）**
   - 中心频率：50Hz或60Hz（根据地区）
   - 带宽：1-2 Hz
   - 作用：去除工频干扰
   - 类型：双T型、状态变量型

**滤波器参数选择：**
```
EMG有效频段：20-500 Hz
高通截止：fHP = 20 Hz
低通截止：fLP = 500 Hz
陷波频率：fNotch = 50/60 Hz
```

### 2.4 模数转换（ADC）

#### 2.4.1 采样定理

**奈奎斯特采样定理（Nyquist-Shannon Sampling Theorem）：**
```
fs ≥ 2 × fmax
```
其中：
- fs：采样率（Hz）
- fmax：信号最高频率成分（Hz）

**EMG信号采样：**
- 信号最高频率：500 Hz
- 理论最低采样率：1000 Hz
- 实际推荐采样率：1000-2000 Hz
- 原因：
  - 留有余量防止混叠
  - 便于后续数字滤波
  - 保留更多信号细节

#### 2.4.2 分辨率

**ADC分辨率：**
- 定义：用多少位（bit）表示模拟电压
- 常见分辨率：12-16 bit

**计算示例（12-bit ADC）：**
```
ADC输入范围：±5V（总范围10V）
分辨率：12 bit = 2^12 = 4096 levels
最小可分辨电压：10V / 4096 = 2.44 mV

放大后EMG范围：±2.5V（增益1000倍）
对应原始EMG：±2.5mV
ADC误差（原始）：2.44mV / 1000 = 2.44 μV
```

**分辨率选择：**
- 12-bit：基本应用（±2.44 μV）
- 16-bit：高精度应用（±0.15 μV）
- 24-bit：科研级（±0.001 μV）

#### 2.4.3 采样率对比

| 采样率 | 可重建频率 | 评价 | 应用 |
|--------|-----------|------|------|
| 500 Hz | 0-250 Hz | 不足 | 不推荐 |
| 1000 Hz | 0-500 Hz | 基本 | 入门学习 |
| 2000 Hz | 0-1000 Hz | 良好 | 标准研究 |
| 4000 Hz | 0-2000 Hz | 冗余 | 特殊需求 |

### 2.6 多通道采集的意义

#### 2.6.1 为什么需要多通道

**1. 肌肉协同分析**
- 不同肌肉在运动中协同工作
- 单通道无法反映整体激活模式
- 例如：握拳动作涉及多块前臂肌肉

**2. 提高分类精度**
- 更多特征信息
- 不同手势的肌肉激活模式不同
- 多通道可以捕捉这些差异

**3. 空间信息**
- 肌肉激活的空间分布
- 疲劳时激活区域的变化
- 肌肉募集顺序

**4. 冗余和鲁棒性**
- 某个通道信号质量差时，其他通道仍可用
- 提高系统可靠性

#### 2.6.2 通道数量选择

**实际应用指导：**

| 通道数 | 应用场景 | 优缺点 |
|--------|----------|--------|
| 1-2 | 基础学习、简单检测 | 简单但信息有限 |
| 4-8 | 手势识别、运动分析 | 平衡性能和复杂度 |
| 16+ | 详细肌肉分析、科研 | 信息丰富但处理复杂 |
| 64-256 | HD-sEMG研究 | 空间分辨率极高 |

**手部动作推荐配置：**
- 基本手势（握拳、张开）：4通道
- 复杂手势（单指运动）：8通道
- 精细动作研究：16+通道

#### 2.6.3 多通道布置策略

**环形布置（Myo Armband方式）：**
```
     ch0
  ch7  ch1
 ch6    ch2
  ch5  ch3
     ch4
```
- 优点：均匀覆盖前臂
- 适用：手势识别

**目标肌肉布置（Targeted）：**
```
通道1: 指浅屈肌
通道2: 指深屈肌
通道3: 拇长展肌
通道4: 桡侧腕屈肌
```
- 优点：针对性强
- 适用：特定动作分析

## 第三部分：EMG信号的干扰与噪声

### 3.1 噪声来源分类

#### 3.1.1 内源性噪声

**1. 其他肌肉的串扰（Crosstalk）**
- 来源：邻近肌肉的电活动
- 特点：与目标肌肉信号相似
- 影响：难以区分，降低定位准确性
- 解决：
  - 合理选择电极位置
  - 使用小间距电极
  - 多通道独立成分分析（ICA）

**2. 心电干扰（ECG Artifacts）**
- 来源：心脏电活动
- 特点：
  - 周期性（心率频率，约1-2 Hz）
  - 幅度大（可达1mV）
  - 主要影响胸部附近肌肉
- 识别：QRS波复合体特征明显
- 解决：
  - 电极远离心脏
  - 自适应滤波
  - 模板减除法

#### 3.1.2 外源性噪声

**1. 工频干扰（Power Line Interference）**
- 来源：电力线辐射（50Hz或60Hz）
- 特点：
  - 固定频率正弦波
  - 幅度可达100μV-1mV
  - 频谱上有明显尖峰
  - 包含谐波（100Hz, 150Hz等）
- 原因：
  - 电极线缆作为天线接收
  - 设备接地不良
  - 周围电器干扰
- 解决方案：
  - 硬件：屏蔽线缆、良好接地
  - 软件：陷波滤波器（50/60Hz）
  - 环境：远离大功率电器

**2. 运动伪影（Motion Artifacts）**
- 来源：
  - 电极相对皮肤移动
  - 导线晃动
  - 身体大幅度运动
- 特点：
  - 低频成分（0-20Hz）
  - 幅度大且不规则
  - 突发性强
- 表现：
  - 基线剧烈漂移
  - 瞬间大幅跳变
- 解决：
  - 固定电极和线缆
  - 高通滤波（≥20Hz）
  - 使用弹力带或胶带固定

**3. 电极-皮肤接触噪声**
- 来源：
  - 接触阻抗高且不稳定
  - 皮肤-电极界面电势变化
  - 汗液影响
- 特点：
  - 低频漂移
  - 高频随机噪声增加
- 解决：
  - 皮肤预处理
  - 使用导电膏
  - 选择合适电极

**4. 仪器噪声**
- 来源：
  - 放大器热噪声
  - ADC量化噪声
  - 电源噪声
- 特点：
  - 宽频带白噪声
  - 幅度相对恒定
- 解决：
  - 使用低噪声器件
  - 良好的电源设计
  - 屏蔽设计

### 3.2 信号质量评估

#### 3.2.1 信噪比（SNR）

**定义：**
```
SNR(dB) = 10 × log10(P_signal / P_noise)
```

**计算方法：**

1. **时域法**
```python
# 假设静息期只有噪声，激活期有信号+噪声
noise = signal[rest_period]
signal_plus_noise = signal[active_period]

P_noise = np.var(noise)
P_signal = np.var(signal_plus_noise) - P_noise
SNR_dB = 10 * np.log10(P_signal / P_noise)
```

2. **频域法**
```python
# 在频谱上区分信号和噪声频段
signal_band_power = integrate_power(20, 150)  # Hz
noise_band_power = integrate_power(300, 450)  # Hz
SNR_dB = 10 * np.log10(signal_band_power / noise_band_power)
```

**质量标准：**
- SNR > 25 dB：优秀
- SNR 15-25 dB：良好
- SNR 10-15 dB：可接受
- SNR < 10 dB：较差，需要改善

#### 3.2.2 信号完整性检查

**检查清单：**

1. **幅度检查**
   - 静息期：<50 μV
   - 激活期：>100 μV
   - 峰值：<5 mV（避免饱和）

2. **频谱检查**
   - 主要能量在20-500 Hz
   - 无明显的50/60Hz尖峰
   - 无异常的低频漂移

3. **时域特性**
   - 零均值（长时间平均）
   - 类高斯分布
   - 无基线大幅漂移

4. **通道一致性**
   - 各通道幅度在合理范围
   - 无某个通道异常（断线或短路）

## 第四部分：实践指导

### 4.1 首次使用设备指南

#### 4.1.1 开箱与连接

**步骤1：检查设备**
- 主机/采集板
- 电极（数量是否足够）
- 线缆
- 电源/电池
- 导电膏/酒精棉片

**步骤2：安装软件**
- 设备厂商提供的驱动
- 数据采集软件
- Python库（pyserial等）

**步骤3：连接设备**
- USB连接（最常见）
- 确认设备被识别（设备管理器）
- 记录端口号（如COM3）

#### 4.1.2 第一次采集信号

**准备工作：**

1. **皮肤准备**
```
1. 选择前臂掌侧干净区域
2. 酒精棉片擦拭（去油）
3. 轻轻打磨（降低阻抗）- 可选
4. 再次酒精擦拭
5. 等待皮肤干燥
```

2. **电极放置**
```
双极配置：
- 电极1：前臂中段，肌腹中央
- 电极2：电极1下方2cm处
- 参考电极：手腕处骨性标志（桡骨茎突）
```

3. **检查连接**
```
- 电极与皮肤贴合良好
- 导线无明显晃动
- 设备指示灯正常
```

**采集测试：**

```python
# 基本测试流程
1. 启动采集软件
2. 观察静息状态信号（应该平稳）
3. 做简单手势（握拳）
4. 观察信号幅度变化（应该明显增大）
5. 返回静息（幅度应恢复）
```

**常见问题：**

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 信号平坦无变化 | 电极未连接 | 检查连接 |
| 信号异常大 | 电极短路 | 重新贴电极 |
| 50Hz明显尖峰 | 接地不良 | 接地或陷波滤波 |
| 基线漂移严重 | 电极松动 | 重新固定电极 |
| 信号饱和 | 增益过大 | 降低增益 |

#### 4.2.1 实验一：静息vs激活对比

**目的：**
观察肌肉静息和激活的EMG差异

**步骤：**
1. 采集10秒静息期数据
2. 采集10秒握拳激活数据
3. 保存数据
4. 对比分析

**预期结果：**
- 静息期RMS：<50 μV
- 激活期RMS：>200 μV
- 幅度比：>4倍

**分析代码：**
```python
rest_rms = np.sqrt(np.mean(rest_signal ** 2))
active_rms = np.sqrt(np.mean(active_signal ** 2))
ratio = active_rms / rest_rms
print(f"激活/静息比值: {ratio:.2f}")
```

### 4.3 问题诊断与解决

#### 4.3.1 信号质量差

**症状：**
- 噪声大
- SNR < 10 dB
- 难以区分激活和静息

**排查顺序：**
1. 检查电极接触
   - 重新贴电极
   - 增加导电膏
2. 检查皮肤阻抗
   - 重新清洁
   - 轻微打磨
3. 检查环境干扰
   - 远离电器
   - 改善接地
4. 调整滤波参数
   - 增强陷波滤波
   - 调整带通范围

#### 4.3.2 信号饱和

**症状：**
- 信号被截断（平顶）
- 幅度达到ADC上限

**解决：**
1. 降低放大器增益
2. 调整电极位置（避开强信号区域）
3. 提醒受试者减小力量

#### 4.3.3 通道间差异大

**症状：**
- 某些通道信号明显强/弱

**原因与解决：**
1. 电极位置差异 → 重新定位
2. 肌肉大小不同 → 正常现象
3. 接触阻抗不一致 → 改善接触
