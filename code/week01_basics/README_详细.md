## 核心知识点总结

### 必须掌握的概念

**生理学基础**：
- 运动单位 = 1个运动神经元 + 其支配的所有肌纤维
- 动作电位：细胞膜去极化和复极化过程
- EMG = 多个运动单位动作电位的叠加

**信号特性**：
- 幅度：50μV - 5mV（静息→强力收缩）
- 频率：20-500 Hz，能量集中在50-150 Hz
- 特征：随机、突发、零均值、非平稳

**采集系统**：
- 电极（表面/针式）→ 放大器（增益1000-10000）
- → 滤波器（20-500Hz带通）→ ADC（≥1000Hz采样）
- → 计算机（数字处理）

**噪声类型**：
1. 工频干扰：50/60Hz正弦波，用陷波滤波器
2. 运动伪影：<20Hz低频漂移，用高通滤波器
3. 电极噪声：高频随机，改善电极接触
4. 心电干扰：周期性，远离心脏

**时域特征（4个核心）**：
1. MAV：平均绝对值，活动强度
2. RMS：均方根，信号能量
3. WL：波形长度，复杂度
4. ZC：过零率，频率信息

**频域特征（2个核心）**：
1. MNF：平均频率，疲劳时下降
2. MDF：中值频率，疲劳检测指标

### 关键公式

```
奈奎斯特采样定理：fs ≥ 2 × fmax
EMG需要：fs ≥ 2 × 500Hz = 1000 Hz

MAV = (1/N) × Σ|x[i]|
RMS = sqrt((1/N) × Σx[i]²)
MNF = Σ(f[i] × P[i]) / Σ(P[i])
SNR(dB) = 10 × log10(P_signal / P_noise)
```
