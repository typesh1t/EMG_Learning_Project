# 第2周：EMG设备认识

## 学习目标
- 了解EMG采集设备的各个组件
- 理解采样率、增益、带宽等重要参数
- 认识常见的EMG设备

## 本周内容

### 1. 设备组件详解
运行 `device_components.py` 了解各组件的功能

### 2. 设备参数理解
运行 `sampling_demo.py` 理解采样率的重要性

### 3. 多通道采集
运行 `multichannel_demo.py` 了解多通道的意义

## 文件说明
- `device_components.py` - 设备组件交互式演示
- `sampling_demo.py` - 采样率对比演示
- `multichannel_demo.py` - 多通道采集概念
- `device_comparison.md` - 常见设备对比
- `assignment2_device_quiz.md` - 作业2：设备知识测试

## 实践任务
1. 运行所有演示脚本
2. 完成设备知识测试
3. 如果有EMG设备，熟悉其使用方法

## 作业
1. 完成 `assignment2_device_quiz.md` 中的问题
2. 查找并了解你将使用的EMG设备的参数
3. 回答：为什么需要多通道采集？举例说明
